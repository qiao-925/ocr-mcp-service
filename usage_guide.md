# OCR MCP Service 使用指南

## 🚀 快速开始

### OCR识别

使用自然语言指令即可调用OCR工具识别图片：

```
请使用OCR工具识别这张图片：图片路径
```

### 示例Prompt模板增强

如需完整的分析流程（OCR识别 + 视觉分析 + 内容总结），可使用示例Prompt模板：

**单张图片：**
```
使用示例prompt模板识别这张图片：图片路径
```

**批量处理：**
```
使用示例prompt模板识别这个文件夹：文件夹路径
```

---

## 📝 功能说明

- **OCR工具**：自动选择合适的OCR引擎进行文字识别
- **示例Prompt模板**：提供完整的分析流程，包括OCR识别、视觉分析和内容总结
- **结果保存**：所有分析结果自动保存在图片所在目录
