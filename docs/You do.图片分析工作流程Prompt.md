# 图片分析工作流程 Prompt

## 📋 任务目标

对图片进行多维度分析，结合 OCR 原文识别、Agent 内容总结和 Cursor 视觉模型，进行全面的数据分析。

## 🔄 工作流程

### 第一步：OCR 原文识别

**使用工具：** `recognize_image_paddleocr` 或 `recognize_image_easyocr`

**操作步骤：**
1. 识别图片中的文字内容
2. 获取文本、文本框坐标、置信度等信息
3. 记录识别结果的技术指标（处理时间、置信度、文本块数量等）

**Prompt 示例：**
```
请使用 recognize_image_paddleocr 工具识别这张图片：[图片路径]
```

### 第二步：视觉模型分析

**使用 Cursor 内置视觉识别能力：**
- 分析图片的视觉元素（布局、结构、图表、表格等）
- 识别图片中的非文本信息（图像、图表、表格结构）
- 理解图片的整体内容和上下文

**Prompt 示例：**
```
请分析这张图片的视觉内容：
- 图片的整体布局和结构
- 是否有图表、表格或其他视觉元素
- 图片的主题和风格
- 视觉元素与文本的关系
```

### 第三步：内容总结与分析

**结合 OCR 识别结果和视觉分析：**

**分析维度：**

1. **文本内容分析**
   - 提取关键信息
   - 识别主题和核心观点
   - 分析文本结构和逻辑关系

2. **视觉结构分析**
   - 分析版面布局
   - 识别标题、段落、列表等结构
   - 理解视觉层次和信息组织方式

3. **数据提取与分析**
   - 提取结构化数据（如表格数据）
   - 识别关键指标和数值
   - 分析数据之间的关系

4. **综合总结**
   - 结合 OCR 文本和视觉分析
   - 生成全面的内容总结
   - 提供关键洞察和建议

## 📝 完整 Prompt 模板

```
请对以下图片进行全面的分析：

1. **OCR 识别**
   - 使用 recognize_image_paddleocr 识别图片中的文字
   - 记录识别置信度和技术指标

2. **视觉分析**
   - 分析图片的整体布局和视觉结构
   - 识别图表、表格、图像等非文本元素
   - 理解视觉层次和信息组织方式

3. **内容总结**
   - 提取关键信息和核心观点
   - 分析文本的逻辑结构和主题
   - 识别重要的数据和指标

4. **综合分析**
   - 结合 OCR 文本和视觉分析结果
   - 生成全面的内容总结
   - 提供关键洞察和发现

图片路径：[图片路径]
```

## 🔧 技术要点

### OCR 识别注意事项
- 中文文档优先使用 `recognize_image_paddleocr`
- 多语言文档使用 `recognize_image_easyocr`
- 注意识别置信度，低置信度结果需要人工复核

### 视觉分析要点
- 充分利用 Cursor 的视觉模型能力
- 关注图片的整体结构和布局
- 识别非文本的视觉元素

### 综合分析策略
- 将 OCR 文本与视觉分析结果交叉验证
- 识别文本和视觉信息之间的关联
- 生成结构化的分析报告

## 📊 输出格式建议

```markdown
## 图片分析报告

### 1. OCR 识别结果
- **识别文本**：[文本内容]
- **置信度**：[数值]
- **处理时间**：[时间]
- **文本块数量**：[数量]

### 2. 视觉分析
- **整体布局**：[描述]
- **视觉元素**：[图表、表格等]
- **结构特点**：[描述]

### 3. 内容总结
- **核心主题**：[主题]
- **关键观点**：[观点列表]
- **重要信息**：[信息列表]

### 4. 数据分析
- **结构化数据**：[数据]
- **关键指标**：[指标]
- **洞察发现**：[发现]
```

## 🚀 优化建议

1. **多引擎对比**：对于重要图片，可以使用多个 OCR 引擎对比识别结果
2. **置信度阈值**：设置置信度阈值，低于阈值的结果需要特别标注
3. **批量处理**：对于多张图片，可以批量处理并生成对比分析
4. **结果验证**：结合视觉分析验证 OCR 识别结果的准确性

## 📌 注意事项

- OCR 识别可能受图片质量、光线、角度等因素影响
- 视觉模型分析结果可能与 OCR 文本有差异，需要综合判断
- 对于复杂布局的图片，可能需要分区域分析
- 重要数据建议人工复核
