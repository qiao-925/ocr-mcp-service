# 图片分析 Prompt 模板

## 🎯 快速使用模板

### 基础模板

```
请对这张图片进行全面的分析：

1. **OCR 识别**：使用 recognize_image_paddleocr 识别图片中的文字内容
2. **视觉分析**：分析图片的整体布局、视觉元素和结构
3. **内容总结**：提取关键信息、核心观点和重要数据
4. **综合分析**：结合 OCR 文本和视觉分析，生成全面的分析报告

图片路径：[图片路径]
```

### 详细分析模板

```
请按照以下步骤分析这张图片：

**第一步：OCR 原文识别**
- 使用 recognize_image_paddleocr 工具识别图片文字
- 记录识别置信度、处理时间等技术指标
- 提取完整的文本内容

**第二步：视觉模型分析**
- 分析图片的整体布局和视觉结构
- 识别图表、表格、图像等非文本元素
- 理解视觉层次和信息组织方式
- 分析图片的主题和风格

**第三步：内容总结**
- 提取关键信息和核心观点
- 分析文本的逻辑结构和主题
- 识别重要的数据和指标
- 理解内容的上下文和背景

**第四步：数据分析**
- 提取结构化数据（如表格数据）
- 识别关键指标和数值
- 分析数据之间的关系和趋势
- 提供关键洞察和建议

**第五步：综合报告**
- 结合 OCR 文本和视觉分析结果
- 生成全面的内容总结
- 标注识别置信度较低的部分
- 提供后续分析建议

图片路径：[图片路径]
```

## 🔧 技术增强模板

### 高精度分析模板

```
请使用多引擎对比的方式分析这张图片：

1. **多引擎 OCR 识别**
   - 使用 recognize_image_paddleocr（中文优化）
   - 使用 recognize_image_easyocr（多语言支持）
   - 对比两个引擎的识别结果和置信度

2. **视觉深度分析**
   - 分析图片的整体布局和视觉结构
   - 识别所有视觉元素（文字、图表、图像、装饰等）
   - 理解视觉层次和信息组织方式
   - 分析图片的视觉风格和设计特点

3. **内容综合分析**
   - 交叉验证 OCR 识别结果
   - 提取关键信息和核心观点
   - 识别结构化数据和非结构化信息
   - 分析内容的逻辑关系和上下文

4. **质量评估**
   - 评估 OCR 识别的准确性
   - 标注低置信度区域
   - 提供改进建议

图片路径：[图片路径]
```

## 🚀 使用技巧

1. **明确目标**：根据分析目标选择合适的模板
2. **指定路径**：使用绝对路径确保图片能被正确找到
3. **多引擎对比**：重要图片建议使用多个 OCR 引擎
4. **结果验证**：结合视觉分析验证 OCR 结果
5. **结构化输出**：使用统一的输出格式便于后续处理
